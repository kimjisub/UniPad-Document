# Structure of the UniPack

The `UniPack`, is the project file of `UniPad`, has contained an individual structure unlike the files using .als in Ableton Live. The UniPack uses `.zip` format.

The unipack largely consists of **5 files**.

|Name|Format|Content|
|------|---|---|
|info|ğŸ“„file|unipack explanation and arguments
|sounds|ğŸ“folder|the folder holding sound file|
|key sound|ğŸ“„file|maps the sound file to the button|
|keyLED|ğŸ“folder|the LED folder holding event files|
|autoPlay|ğŸ“„file|saves voice record to automatically press the button|

:::info

The **files** used in unipack do not have an **extension** except a sound file.
