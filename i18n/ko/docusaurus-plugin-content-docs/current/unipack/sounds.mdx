각 버튼에 사용될 샘플링된 음악 파일들을 담습니다.

---

## Directions

- .wav 확장자를 사용해야 하며, PCM 16bit 코덱을 권장합니다.
- 사운드파일의 이름은 **영어**, **숫자**만 포함해야 하며, **한글이나 공백문자**는 포함될 수 없습니다.
- 사운드는 보통 **6초**까지 지원합니다.

``` text title="가능한 예시"
1_01.wav
drum1.wav 
```

``` text title="불가능한 예시"
1 01.wav
드럼1.wav
sound.mp3
```
